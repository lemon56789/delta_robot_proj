# Plan

## Goal
팀원들이 제공한 원본 이력과 선호 주제를 팀원 프로필 문서에 반영해, 외부 AI가 역할뿐 아니라 경험 기반 강점과 주의점을 함께 이해할 수 있게 한다.

## Changes
- 각 팀원 프로필 문서에 원본 경험 이력을 반영한다.
- 약어를 풀어서 학부생 연구 프로그램, 진동동역학실습, 열전달 및 열유체실습 등으로 명시한다.
- 강점, 약할 수 있는 분야, 선호 주제를 분리해 외부 AI 입력용 문서로서 활용도를 높인다.
- 오늘 작업 내역을 Daily Note에 추가한다.

## Impact
- 외부 AI가 팀원별 배경지식과 선호 분야를 더 정확히 반영한 답변을 생성할 수 있다.
- 역할 중심 1차 프로필이 경험 기반 프로필로 확장된다.

## Risk
- 원본 기록은 자기보고 형식이라 세부 숙련도는 실제와 차이가 있을 수 있다.
- 추후 팀원 경험이 추가되면 문서 재정비가 필요하다.

## Validation
- 각 프로필 문서에 원본 경험, 사용 툴, 강점/주의점, 선호 주제가 반영되었는지 확인한다.
- 약어가 모두 풀어써졌는지 확인한다.
- `docs/daily_notes/2026-03-22.md`에 변경 기록이 추가되었는지 확인한다.
