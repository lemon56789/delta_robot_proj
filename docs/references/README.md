# References

이 디렉터리는 프로젝트 구현과 문서 작업에서 참고하는 외부 자료를 보관하는 용도다.
여기에 저장된 논문, 슬라이드, 메모는 `reference`이며, 프로젝트의 공식 채택안이나 Source of Truth를 의미하지 않는다.

## Purpose
- IK, dynamics, simulation, sensing 관련 참고 논문을 보관한다.
- 구현 시 필요한 수식 형태나 배경 설명의 출처를 정리한다.
- 프로젝트 문서에서 아직 채택하지 않은 방법도 참고 자료로 유지할 수 있다.

## Scope
- PDF, 이미지, 요약 메모 등 참고 자료를 둘 수 있다.
- 현재 프로젝트 기준이나 인터페이스를 바꾸는 근거 문서로 자동 승격되지 않는다.

## File Naming
- 파일명은 짧고 식별 가능하게 정리한다.
- 가능하면 `year_author_short-title.pdf` 형식을 권장한다.

예시:
- `2012_smith_delta-robot-ik.pdf`
- `2020_lee_planar-calibration-note.pdf`

## Note
- IK 관련 논문을 여기에 넣는 것은 `2atan(t)` 또는 다른 방식을 프로젝트 채택안으로 확정했다는 뜻이 아니다.
- 실제 채택 기준은 별도 문서 또는 합의로 관리한다.
