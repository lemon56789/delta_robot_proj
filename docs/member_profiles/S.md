# S Profile

## Role
동역학 해석, 진동 분석 담당.

## AI Prompt Seed
S는 동역학 해석과 진동 분석 관점에서 답변을 원한다. 원인과 영향 경로를 먼저 설명하고, 제어 설계와 실물 제작의 연결 관계를 함께 정리하라.

## Background
- 학부생 연구 프로그램에서 구형 로봇 주제를 수행했다.
- 해당 연구에서 동역학 해석 도구 RecurDyn과 MATLAB, Simulink를 사용해 본 경험이 있다.
- 진동동역학실습에서 Arduino, Autodesk, 3D 프린팅을 활용한 제작 경험이 있다.
- 제어 설계 관련 경험이 있어 로봇이나 강체 기반 실물 제작 주제에 선호가 있다.

## Scope
- 델타 로봇의 동역학 거동 분석
- 진동 특성 파악 및 오차 원인 해석
- 시뮬레이션 결과와 기구 거동의 정합성 검토
- 제어와 기구 응답이 연결되는 구간의 해석 지원

## Strengths
- 동역학 해석 및 진동 분석 경험
- 제어 설계 관점과 실물 제작 관점을 함께 연결할 수 있음
- 로봇 시스템의 거동을 시뮬레이션과 제작 관점에서 함께 볼 수 있음

## Weaknesses Or Cautions
- 제공된 기록 기준으로 데이터 기반 인공지능 모델링 경험은 핵심 강점으로 보이지 않는다.
- 답변 시 제어 및 동역학 중심 설명이 유효하지만, 학습 모델 상세 설계는 별도 검토가 필요할 수 있다.

## Common AI Support Needs
- 동역학 모델링 절차 정리
- 진동 원인 가설 정리 및 비교
- 실험 결과 해석 프레임 정리
- 시뮬레이션 조건 비교 기준 작성
- 실물 제작과 해석 결과를 연결하는 설명 정리

## Response Priorities
1. 진동과 동역학 관점에서 원인과 영향 경로를 먼저 설명한다.
2. 제어 설계와 실제 기구 응답의 연결 관계를 분명히 한다.
3. 실험 조건, 파라미터, 비교 기준을 분리해서 정리한다.
4. 재현 가능한 분석 절차와 기록 항목을 포함한다.
