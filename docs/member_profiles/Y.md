# Y Profile

## Role
아두이노 제어 담당.

## AI Prompt Seed
Y는 아두이노 기반 제어와 실제 테스트 관점에서 답변을 원한다. 제어 입력과 출력, 측정값 흐름, 파라미터 조정 절차를 실험 중심으로 설명하라.

## Background
- 자율주행경진대회와 자율주행캡스톤디자인 강의를 수강했다.
- ROS 2를 이용한 자율주행 알고리즘을 공부한 경험이 있다.
- Arduino를 이용한 주행 알고리즘 구현과 실제 주행 테스트에서 파라미터 수정을 담당했다.
- 진동동역학실습에서는 Arduino 프로그래밍 전체를 맡았고, PID 제어를 통해 RPM 제어를 수행했다.
- 미래모빌리티공학, 제어공학기초 수업을 수강해 제어에 대한 기본 지식을 갖추고 있다.
- 학부생 연구 프로그램이나 산학협력 현장실습 경험은 없어 상대적으로 실무형 연구 경험은 적을 수 있다.
- 자율주행과 로봇 관련 주제를 선호하며, 현재 기구학과 로봇공학입문 수업을 수강 중이라 로봇의 동작 제어에 관심이 있다.

## Scope
- 목표 궤적을 모터 명령으로 변환하는 제어 로직 구현
- 실기기 제어 흐름과 펌웨어 동작 검토
- 측정 가능한 모터 상태 데이터 확보
- PID 기반 속도 및 동작 제어 조정

- 자율주행 알고리즘과 로봇 동작 제어에 대한 높은 관심

## Weaknesses Or Cautions
- 학부생 연구 프로그램이나 산학협력 현장실습 경험이 없어 장기 연구 운영 경험은 상대적으로 적을 수 있다.
- 고급 모델링이나 해석 중심 질문은 배경 수준을 고려해 단계적으로 설명하는 편이 좋다.

2. 실제 하드웨어 적용 시 안전성과 검증 절차를 먼저 고려한다.
3. 파라미터 조정 절차를 실험 중심으로 설명한다.
4. 로그 수집 항목과 테스트 조건을 빠뜨리지 않는다.
