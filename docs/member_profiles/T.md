# T Profile

## Role
기구 설계 담당.

## AI Prompt Seed
T는 기구 설계와 시스템 제어의 연결 관점에서 답변을 원한다. 구조, 조립성, 설계 변경 영향과 제어 요구를 함께 고려해서 설명하라.

## Background
- 열전달 및 열유체실습에서 Ansys를 사용해 해석을 진행한 경험이 있다.
- 진동동역학실습 프로젝트에서는 Arduino 프로그래밍을 담당했다.
- 두 수업에서 다룬 프로그램은 어느 정도 활용 가능하다.
- 학부생 연구 프로그램 경험은 아직 없어 해당 부분에서는 상대적으로 부족할 수 있다.
- 현재 기구학 수업을 수강 중이며, 기계 시스템의 구조와 동작에 관심이 있다.
- 선호하는 주제는 기구 설계나 시스템 제어 관련 분야다.

## Scope
- 델타 로봇 기구 구조 설계
- 부품 배치, 링크 구조, 조립성 검토
- 제작 가능한 CAD 자산 정리
- 구조와 제어 요구를 함께 고려한 설계 검토

## Tools
- Ansys

## Strengths
- 기계 시스템 구조와 동작에 대한 관심이 높음
- 해석 도구와 Arduino 프로그래밍을 모두 다뤄본 경험이 있음
- 기구 설계와 시스템 제어를 연결해서 볼 수 있음

## Weaknesses Or Cautions
- 학부생 연구 프로그램 경험이 없어 연구 운영이나 문헌 기반 탐색은 단계적으로 지원하는 편이 좋다.
- 현재 정보 기준으로 심화 구조 해석이나 학습 모델링은 핵심 강점으로 보이지 않는다.

## Common AI Support Needs
- 기구 설계 검토 항목 정리
- 조립과 제작 시 고려사항 목록화
- 설계 변경 영향 범위 파악
- 기구 오차가 시스템 성능에 주는 영향 정리
- 구조와 제어 요구를 함께 만족하는 설계 기준 정리

## Response Priorities
1. 기구 구조와 제작 가능성 관점에서 설명한다.
2. 설계 변경이 다른 서브시스템에 주는 영향을 함께 정리한다.
3. 필요한 경우 해석 결과와 제어 요구를 연결해 설명한다.
4. 부품, 조립, 공차 관점의 확인 항목을 포함한다.
